# Fcitx输入法

```bash
$ sudo rm -rf ~/.config/fcitx # 删掉旧的配置 
$ sudo apt-get install fcitx
$ sudo apt-get install fcitx-fontend-gtk2 fctix-fontend-gtk3
$ sudo apt-get install fcitx-frontend-all 
$ sudo apt-get install fcitx-baidupinyin # 百度拼音输入法
$ sudo apt-get install fcitx-ui-qimpanel
$ fcitx-diagnose 	# 打印出当前fcitx输入法的所有配置信息
$ im-config      	# 重新设置输入法
$ fcitx-configtool 	# 配置fcitx输入法
```
