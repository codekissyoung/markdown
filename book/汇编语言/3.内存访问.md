# DS
- DS 存放要访问的数据所在内存的段地址
- 读取 10000H 单元的数据到 al
- 8086 cpu不支持直接将数据送入寄存器的操作，所以需要中转,先送入bx,再由bx送入ds
    ```x86asm
    mov bx,1000H
    mov ds,bx
    mov al,[0]
    ```
- 寄存器中数据写入到内存 , 将al里的数据写入到10000H
    ```x86asm
    mov bx,1000h
    mov ds,bx
    mov [0],al
    ```

# 字
- 每次mov指令读取的都是一个字长
