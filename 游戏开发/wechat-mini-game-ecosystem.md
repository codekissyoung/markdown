# 微信小游戏生态笔记

> 记录于 2024-12 | 为独立游戏开发做准备

## 微信小游戏 vs Steam 独立游戏

两个完全不同的生态，宣发逻辑不能混用。

| 维度 | 微信小游戏 | Steam独立游戏 |
|:-----|:----------|:-------------|
| 用户获取 | 被动触达（分享链接点进来） | 主动搜索（愿望单、商店浏览） |
| 核心增长 | 裂变分享、买量 | 内容营销、口碑传播 |
| 用户行为 | 玩完就走，留存是挑战 | 主动下载，付费意愿高 |
| 变现模式 | IAA（广告）为主 | 买断制/IAP |
| 开发周期 | 短平快，快速迭代 | 长周期，内容驱动 |
| 技术栈 | 小游戏引擎（Cocos等） | Unity/Unreal/Godot |

## 微信小游戏的增长方式

### 1. 分享裂变（核心）

游戏内设计激励分享机制：
- 分享得复活机会
- 分享解锁新关卡
- 分享获得金币/道具

好友点开链接 → 新用户。**裂变系数**决定能否自然增长。

### 2. 小游戏广告（买量）

微信广告平台投放，按点击/转化付费。IAA模式必备：
- 花钱买用户
- 用户看广告
- 广告收入 > 买量成本 = 正向循环

### 3. 视频号

微信生态内的流量入口：
- 游戏实况/攻略视频
- 挂载小游戏链接
- 适合有内容产出能力的团队

### 4. 私域（冷启动）

- 朋友圈转发
- 微信群分享
- 适合初期验证，不可规模化

## 变现模式决定宣发策略

### IAA（广告变现）

特点：需要大量用户，ARPU低
- 休闲游戏为主（消消乐、跑酷、合成类）
- 宣发重点：买量 + 裂变
- 关键指标：LTV > CAC（用户终身价值 > 获客成本）

### IAP（内购变现）

特点：需要精准付费用户，ARPU高
- 中重度游戏、社交游戏
- 宣发重点：社群运营、内容营销
- 关键指标：付费率、ARPPU

## Steam独立游戏宣发清单（备忘）

微信生态暂时用不上，但将来做Steam可以参考：

1. 多平台发视频（B站、抖音、YouTube）
2. 尽早上架Steam商店（积累愿望单）
3. 参加Steam新品节（免费曝光，需要Demo）
4. 合理设置折扣
5. 参与Steam促销活动
6. 开通小黑盒商店（国内玩家社区）
7. 发放CDKEY（给主播/媒体试玩）
8. 对接Steam鉴赏家

**注意**：这些是"渠道清单"，不是"成功策略"。游戏本身质量才是根本。

## 独立开发者的务实路径

### 阶段一：跑通流程

做一个极简休闲小游戏：
- 点击类（消灭星星、2048变种）
- 跑酷类（一键操作）
- 合成类（合成大西瓜类型）

目标不是赚钱，是跑通：**Cocos开发 → 微信小游戏导出 → 审核上线 → 接广告SDK**

### 阶段二：验证玩法

在阶段一基础上迭代：
- 观察数据（留存、分享率、广告观看率）
- 调整玩法和数值
- 学习游戏设计

### 阶段三：正式产品

有了经验再做"认真"的项目。

## 关键认知

1. **宣发救不了烂游戏** - 好游戏自己会传播
2. **完成度是最大瓶颈** - 大部分项目死在"做不完"
3. **先跑通再优化** - 不要在第一个项目上追求完美
4. **微信和Steam是两个世界** - 策略不能混用
