# 《你必须知道的495个C语言问题》

## `char *p1, p2;`

- 这样声明, p1 是指针, p2 是`char`类型
- 声明两个指针, 使用 `char *p1, *p2;`

## `char *p; *p = malloc(10);`

- 正确使用指针`p = malloc(10);`

## `extern int f();`

- `extern`表明`f()`函数可能定义在另一个源文件中

## 必须先定义一个`typedef类型`，然后再使用它

```c
// 错误
typedef struct{
    char *item;
    NODEPTR next;
} *NODEPTR;
// 正确
typedef struct node{
    char *item;
    struct node *next;
} *NODEPTR;
```

## 如何定义一对互相引用的结构?

```c
// 错误
typedef struct{
    int afield;
    BPTR bpointer;
} *APTR;
typedef struct{
    int bfield;
    APTR apointer;
} *BPTR;
// 正确
struct b;
typedef a{
    int afield;
    struct b *bpointer;
};
typedef b{
    int bfield;
    struct a *apointer;
};
typedef struct a *APTR;
typedef struct b *BPTR;
```
