# Linux 进程相关函数

## 进程

```c
pid_t getpid(void);  // 获取当前进程 ID
pid_t getppid(void); // 获取当前进程的 父进程ID
uid_t getuid(void);  // 当前进程的 用户ID
uid_t geteuid(void); // 当前进程的 有效用户ID
gid_t getgid(void);  // 当前进程的 组ID
gid_t getegid(void); // 当前进程的 有效组ID
```

## 创建新进程

```c
if( ( pid_t pid = fork() ) > 0 )
{
    // 父进程
}
else if( pid == 0 )
{
    // 子进程
}
else
{
    // 创建新进程失败
}
```
