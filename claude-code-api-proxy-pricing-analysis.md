# Claude Code API代理服务定价分析

## 背景

Claude Code 是 Anthropic 官方的命令行 AI 编程助手，需要消耗 API 额度。国内用户通常通过第三方代理服务使用，本文分析某代理平台的套餐定价策略。

## 套餐对比

| 套餐 | 价格 | 额度 | opus支持 | 特点 |
|:-----|:-----|:-----|:--------:|:-----|
| PAYGO | ¥66/年 | 165$ cc / 330$ codex | ✅ | 入门按量付费 |
| PLUS | ¥198/月 | 20$/天，可重置2次 | ❌ | 每日上限40$ cc |
| PRO | ¥398/月 | 60$/天，可重置2次 | ❌ | 每日上限120$ cc，推荐标签 |
| FREE | ¥88/年 | 20刀总上限 | ❌ | 消耗倍率高(cc 10倍) |
| PAYGO高级 | ¥666/年 | 一次性1988$ | ✅ | 大额按量付费 |

### 关键参数说明

**消耗倍率**：
- codex 0.5倍：实际可用额度翻倍
- cc (Claude Code) 1倍：正常消耗
- FREE套餐 cc 10倍/codex 4倍：消耗速度快

**额度重置机制**（PLUS/PRO）：
- 每天可重置2次
- 重置间隔 ≥ 5小时
- 理论最大：每日上限 × 3

## 套餐选择建议

### 需要 opus 模型

只有两个 PAYGO 套餐支持 opus：

| 套餐 | 价格 | 实际codex额度 | 性价比 |
|:-----|:-----|:-------------|:-------|
| ¥66 PAYGO | ¥66/年 | 330$ | 5$/¥ |
| ¥666 PAYGO | ¥666/年 | ~4000$ | 6$/¥ |

**推荐 ¥666 PAYGO**：一次性大额度，不用担心中途用完。

### 不需要 opus 模型

对比月费套餐和年费套餐：

```
PLUS 年成本：¥198 × 12 = ¥2376
PRO 年成本：¥398 × 12 = ¥4776
¥666 PAYGO：¥666（一次性1988$）
```

**结论**：即使不用 opus，¥666 PAYGO 依然是最佳选择。

### 轻度用户

¥66 PAYGO 或 ¥88 FREE：
- ¥66 性价比更高，支持 opus
- ¥88 FREE 消耗倍率太高，不推荐

## 不推荐的选择

### PLUS/PRO 套餐问题

1. **月费制累积成本高**
2. **不支持 opus 模型**
3. **每日上限限制**：重度使用会卡额度
4. **重置机制繁琐**：需要等待5小时间隔

### FREE 套餐问题

1. **消耗倍率过高**：cc 10倍意味着 20刀只相当于 2刀实际额度
2. **总额度太低**：20刀上限，几天就用完
